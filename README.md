# 4 Day Workout Tracker

A static GitHub Pages workout tracker for iPhone home screen use.

## Upload to GitHub

1. Unzip this folder.
2. Open your GitHub repository in Safari or desktop browser.
3. Choose **Add file** then **Upload files**.
4. Upload all files in this folder: `index.html`, `styles.css`, `app.js`, `manifest.json`, `sw.js`, `icon.svg`, `README.md`.
5. Commit changes.
6. Go to **Settings > Pages**.
7. Choose **Deploy from a branch**, branch `main`, folder `/root`.
8. Open the live GitHub Pages link in Safari and choose **Share > Add to Home Screen**.

## Notes

Your workout data is stored privately in browser local storage. Use **Export backup** regularly.
